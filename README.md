# MCROSS

Minimal roof cleaning landing page built with Next.js for Vercel.

## What it includes

- One-page black-background landing page
- Green intro screen that gets washed away by a water-rinse animation
- Mobile-friendly layout
- Simple quote-first contact CTA

## Local development

```bash
npm install
npm run dev
```

## Production check

```bash
npm run build
```

## Before going live

Update the placeholder phone number, text link, email, and service area in `src/app/page.tsx`.
